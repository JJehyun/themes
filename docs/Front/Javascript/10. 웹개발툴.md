# 네트워크 항목 디버깅하기

{: .highlight }
> - 개발자 도구 네트워크 탭
>   - 네트워크 탭에 톱니바퀴로 자원을 다운로드하는데 얼마나 걸렸는지 알 수 있음 (`병목이 생기는 지`)
>   - api응답에 대한 자세한 정보도 관리자 도구를 통해 알 수 있다.
>   - 브라우저 응답값으로
>       - platform : `sec-ch-ua-plarform`
>       - 모바일 정보 : `sec-ch-ua-mobile`
